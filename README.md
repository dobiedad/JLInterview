# John Lewis Swift Task

### Dependencies

   cocoapods is needed to build the project. The following pods were used
  - `SwiftyJSON`
  - `Alamofire`
  - `ImageSlideshow`
  - `SDWebImage`

### Install

   1. Clone project
   2. `cd JohnLewisSwiftTask`
   3. `pod install`
   4. Open the generated `JohnLewisSwiftTask.xcworkspace`
